Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito (não semântico) usando a tag <b>

Nesta frase temos um termo em destaque (semântico) usando a tag <strong>

Itálico/Ênfase

Nesta frase temos um termo em itálico (não semântico) usando a tag <i>

Nesta frase temos um termo em ênfase(semântico) usando a tag <em>

Texto marcado

Podemos também selecionar um texto marcado usando a tag <mark>

Outro exemplo de texto marcado

Texto grande e Texto pequeno

Estamos criando um texto grande usando a tag <big> (não semântico/obsoleto) e um texto pequeno usando a tag <small>

Texto deletado

Podemos também usar um texto deletado usando a tag <del>

Texto inserido

Podemos inserir um texto utilizando a tag <ins>

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito e subescrito

Podemos sobreescrever um texto utilitando a tag <sup> x45 + 5 = y

E também subescrever um texto utilizando a tag <sub> H2O
